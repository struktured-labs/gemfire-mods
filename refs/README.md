# External references

A flat index of everything worth bookmarking. Each entry has a one-line description so you can scan.

## Tools

- [Koei Decompress (RHN 1083)](https://www.romhacking.net/utilities/1083/) — Windows; (de)compresses graphics, data, and some text in Gemfire / ROTK III / ROTK IV / Genghis Khan II SNES ROMs. Outputs `.kdf`, `.kcf`, `.dat`.
- [Nightmare (RHN 1307)](https://www.romhacking.net/utilities/1307/) — General-purpose ROM module editor; loads game-specific module files for labeled point-and-click editing.
- [Gemfire SNES Nightmare modules (Yandex Disk)](https://disk.yandex.com/d/vFHEwW7I3YME44) — PleaseYourName's module pack covering all officers, families, and scenarios in SNES Gemfire.

## Hacks

- [Dawn of Ishmeria (RHN 5551)](https://www.romhacking.net/hacks/5551/) — DragonAtma, complete, IPS — prequel mod with rebalanced fifth units.

## Documents

- [DragonAtma's Gemfire notes (.ods, RHN 867)](https://www.romhacking.net/documents/867/) — Mapped data, source for Nightmare modules. **Need to obtain.**
- [DizavidHZ's Scenario Edit Guide](https://gamefaqs.gamespot.com/snes/588352-gemfire/faqs/71083) — In-game save-state trick to author "custom scenarios" without modding the ROM. Mirrored in [`docs/scenario-edit-guide.md`](../docs/scenario-edit-guide.md).
- [DragonAtma's General Stats guide](https://gamefaqs.gamespot.com/snes/588352-gemfire/faqs/50679) — Officer / unit / item stats.
- [admtanaka's full FAQ](https://gamefaqs.gamespot.com/snes/588352-gemfire/faqs/31840) — Walkthrough; canonical reference for prices and item locations.
